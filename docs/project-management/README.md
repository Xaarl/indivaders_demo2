# Project Management Workspace

Date: 2026-06-16

This folder is the PM-facing workspace for sprint planning, decision logs, scope notes, and handoff checklists.

## Start Here

Read in this order:

1. `../product/README.md`
2. `../product/product-definition.md`
3. `../product/product-stages-roadmap.md`
4. `../product/next-sprint-recommendations.md`
5. `../repository/team-handoff.md`

## What Belongs Here

- Sprint plans.
- Product decision logs.
- PM acceptance criteria.
- Stakeholder notes.
- Launch/readiness checklists.
- Meeting summaries that affect scope or priority.

## What Does Not Belong Here

- Source code.
- Design source files.
- Raw research dumps.
- Generated screenshots.
- Private customer data.
- API keys or credentials.
