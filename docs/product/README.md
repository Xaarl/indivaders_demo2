# Indievaders Product Source Of Truth

Audited on 2026-06-16.

This folder is the product source of truth for Indievaders. It defines what the product sells now, what it should become later, what each public and private surface is responsible for, and what engineering/design should build next.

## Canonical Product Decision

Indievaders first sells an interactive Steam Positioning Report for serious solo indie developers and small teams. The report helps teams understand true competitors, unsafe benchmarks, review risks, production reality, price/scope pressure, and next actions before demo, Steam page, Next Fest, outreach, or launch decisions harden.

The long-term product is a private project workspace for one game, with refreshable market intelligence. That future should grow from the paid report, not replace it prematurely.
