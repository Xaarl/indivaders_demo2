## Core Thesis

Refractured has a credible commercial lane if it is positioned as a brutal 2.5D action roguelite, not a generic brawler. The strongest first proof is a focused nightmare micro-run that shows heavy contact, readable pressure, fast retry, and ritual choices that change how the player fights.

Path A remains the recommended first market test: one excellent arena, then a short run loop that proves second-run desire before the team expands content, co-op, or deeper mastery systems. Path B and Path C should be earned by player evidence, not chosen from ambition alone.

The north-star validation metric is Qualified Wishlist Intent from Playable Proof: target players understand the hook, want another run, and would wishlist because of the playable fantasy.
